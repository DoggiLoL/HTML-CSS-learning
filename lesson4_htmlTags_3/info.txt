Практика №1
Задание: Страница профиля пользователя
Описание задачи: Создай простую страницу профиля пользователя, где будут представлены его основные данные и интересы. Страница должна содержать заголовки разного уровня, текстовые абзацы, изображение профиля, списки интересов и контактную форму.

Элементы страницы:

Заголовок страницы: Используй <h1> для названия страницы, например, "Профиль пользователя".

Заголовки разделов: Используй теги <h2> и <h3> для обозначения основных разделов, таких как "О пользователе", "Интересы" и "Контактная информация".

О пользователе:

Используй тег <p> для написания краткой информации о пользователе (например, имя, возраст, род занятий).
Вставь изображение профиля пользователя с помощью тега <img/>.
Интересы пользователя:

Используй тег <h3> для заголовка раздела "Интересы".
Создай упорядоченный список <ol> для основных интересов пользователя, таких как "Чтение", "Путешествия", "Программирование".
Создай неупорядоченный список <ul> для подкатегорий интересов (например, для "Чтения" это могут быть "Фантастика", "Детективы", "Научная литература").
Ссылки на социальные сети:

Используй тег <a> для добавления ссылок на профили пользователя в социальных сетях, таких как Facebook, Instagram, LinkedIn.
Контактная форма:

Добавь форму с тегом <form>, где пользователь может оставить свои контактные данные и сообщение.
Поля формы должны включать:
Имя (<input type="text">)
Адрес электронной почты (<input type="email">)
Сообщение (<textarea>)
Чекбокс для подписки на рассылку новостей (<input type="checkbox">)
Кнопку для отправки формы (<input type="submit">)
Условия:

Используй только HTML для реализации задания.
Обрати внимание на правильную семантику и использование тегов.
